# Tomiko Trade: Car Marketplace Backend

Этот проект представляет собой серверную часть (backend) для автомобильного маркетплейса, разработанную на фреймворке **Django**.

## Основные функции:

- **Управление данными:** Реализована архитектура баз данных для учета автомобилей, пробега и других характеристик.
- **Бизнес-логика:** Интегрированы модули для расчета стоимости и обработки валютных курсов.
- **Обратная связь:** Настроены формы для взаимодействия с пользователями.
- **Технологический стек:** Python, Django, SQLite.

## Статус проекта:

Это учебный командный проект. В данный момент фокус сделан на изучении Django-архитектуры, работы с моделями (ORM) и роутингом запросов. Проект демонстрирует навыки интеграции различных компонентов бэкенда в рамках одного приложения.

---

\*Примечание: Фронтенд-часть проекта использует внешние ресурсы, которые в данный момент могут быть недоступны.

Как запустить локально

    1. Клонируйте репозиторий:
    git clone https://github.com/MrsIgnis/tomiko-trade.git

    2. Установите зависимости:
    pip install -r requirements.txt

    3. Выполните миграции:
    python manage.py migrate

    4. Запустите сервер:
    python manage.py runserver
